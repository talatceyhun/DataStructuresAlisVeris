# DataStructuresAlisVeris
Bu uygulama data structures dersinin 3. projesidir. Komut satırı üzerinden çalışan bu uygulama bir alış veriş sisteminin simülasyonudur. Uygulamada personel girişi ve kullanıcı girişi bulunmaktadır. Personel tarafında sisteme ürün ekleme ürünleri güncelleme vardır. Kullanıcı tarafında ise ürün arama, sepete ürün ekleme ve ürünleri satın alma vardır. Ayrıca kullanıcıya Appriori ve kNN algoritmaları ile satın alınacak ürün önerileri yapılmaktadır. Uygulamanın bir diğer özelliği ise veri setlerini kullanarak kullanıcıların yaptığı yorumların yüzde kaç olumlu olduğunu hesaplayabilmesidir. Kullanılan veri yapıları; heap, tree, hashtable ve genericList'dir.
